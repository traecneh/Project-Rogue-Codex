Purge System

Remove applied special effects or cleanse corrupted innates without confusing the two cleanup paths.

Cleanup Roles

Purge

Special Effect

Use Purge when the item has an applied special effect or imbuement you no longer want.

Cleanse

Corrupted Innate

Use Cleanse when a corrupted item needs its corrupted innate removed.

Recovery

25 Tattered Imbuements

Purging a special effect returns 25 Tattered Imbuements for that effect type.

Finality

No Item Reset

The cleanup target is removed; the item does not reroll, ascend, or reset its other values.

Purge or Cleanse

What Purge Removes

Applied Special Effects

  • Targets the item's removable special effect slot.
  • Best used when an imbuement no longer fits the build.
  • Returns 25 Tattered Imbuements instead of the full Scroll of Imbuement cost.

What Cleanse Removes

Corrupted Innate

  • Targets the corrupted innate effect on corrupted gear.
  • Returns the item toward its normal innate identity.
  • Does not act as a rarity, stat, or max-rarity reroll.

Recovery Rules

Known Return

25 Tattered Imbuements

Purge returns the small tattered-material recovery shown by the system, not the original 250-tatter craft cost.

Tier

No Tier Refund

Do not plan around recovering the original tier roll, scroll, or high-tier result after purging.

Item State

No Item Reset

Purge and Cleanse are cleanup tools; use Re-Roll or Ascend when the item itself needs improvement.

Eligibility

Epic+ Item

Imbuement planning still starts with an Epic+ item, since that is the gear tier that receives crafted imbuements.

Cleanup Flow

  1. 1

    Inspect the item

    Check whether the unwanted line is a special effect, an imbuement, or a corrupted innate.

  2. 2

    Choose the correct tool

    Use Purge for special effects; use Cleanse for corrupted innates.

  3. 3

    Confirm the loss

    Make sure the item is worth keeping after the cleanup, because the removed effect is gone.

  4. 4

    Rebuild if needed

    Return to Imbuements, Craft, Re-Roll, or Ascend if the cleaned item still needs work.

Before You Confirm

Purge is useful when a good item has the wrong special effect, but it is not a full undo. Check the item's base value, rarity ceiling, and build fit before spending the cleanup.
