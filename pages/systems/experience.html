Experience Pool System

Stored levels that build passively and double XP while they are available.

Experience Pool at a Glance

Pool Cap

3.0 levels

The pool caps at 3.0 levels for every character.

Bonus

Double XP

Stored pool doubles XP earned from hunting while pool remains.

Drain

1% = 0.01 pool

Each 1% of a level gained consumes 0.01 pool; a 25% level gain costs 0.25 pool.

Empty

Normal XP

When the pool is empty, XP returns to the normal rate until the pool refills.

Daily Pool Build

Levels 1-89

+3.0 levels per 24 hours

Characters below level 90 rebuild the pool at the full daily rate.

Levels 90+

+1.0 level per 24 hours

High-level characters rebuild more slowly, while the cap remains 3.0 levels.

At Cap

Paused

Passive gain pauses at 3.0 levels until you spend pool through combat XP.

Combat Conversion

Base XP

Kill or Tick XP

XP gain starts with the tick's base XP, then applies the XP multiplier.

Pool Bonus

Matched XP

If pool remains, it adds an equal bonus up to the amount of pool available.

Spend Rate

Percent of Level

Pool consumption is based on how much of the current level the base XP would award.
